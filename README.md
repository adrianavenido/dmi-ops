# Digital Minds Operational Excellence Center

Internal web dashboard for Digital Minds BPO Service Inc. focused on operational visibility, collaboration, and continuous improvement workflows.

## Overview

This app provides a unified interface for operations, HR, talent, and risk workflows used internally across teams. It is not intended for public distribution.

## Tech Stack

- HTML
- Alpine.js
- Tailwind CSS
- Webpack

## Local Development

### Prerequisites

- Node.js 18+

### Install and Run

```bash
npm install
npm run start
```

### Build

```bash
npm run build
```

## Project Structure

- `src/partials/` shared layout blocks (header, sidebar, modals)
- `src/js/router.js` SPA routing
- `src/images/logo/` brand assets and favicon
